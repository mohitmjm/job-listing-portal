# 💼 Job Listing Portal

A full-stack Job Listing Portal built using the MERN stack (MongoDB, Express, React, Node.js).  
This platform allows employers to post jobs and manage applications, while job seekers can browse and apply for jobs.

---

## 🚀 Features

### 🔐 Authentication & Authorization
- User registration (Seeker / Employer)
- Secure login using JWT
- Role-based route protection
- Password hashing using bcrypt

### 👤 Job Seeker
- Browse job listings
- Search jobs
- View detailed job description
- Apply for jobs
- Track application status
- Upload resume
- Manage profile

### 🏢 Employer
- Create job postings
- Update/Delete jobs
- View applicants
- Accept/Reject applications

### 📁 File Upload
- Resume upload using Multer
- File type validation (PDF, DOC, DOCX)
- File size restriction
